# Data Analysis Sales Dashboard

## Project Overview
This project analyzes sales data to identify trends, top-performing products, and regional sales performance. The goal is to generate insights that help businesses make data-driven decisions.

## Tools Used
- Power BI
- SQL
- Excel

## Key Features
- Interactive sales dashboard
- KPI tracking (Revenue, Profit, Sales Growth)
- Product and region performance analysis
- Data cleaning and transformation

## Insights
- Identified top-selling products
- Analyzed regional sales performance
- Detected seasonal sales trends

## Dashboard
Power BI dashboard created to visualize sales metrics and business insights.
